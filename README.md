# React Native Shopping List App

A modern shopping list application built with **React Native, Redux Toolkit, and Expo**, featuring **add, edit, delete, and mark items as purchased**, with persistent storage via AsyncStorage.

---

## üì± Features

- Add new shopping items with name and quantity  
- Edit or delete existing items  
- Mark items as purchased with a toggle switch  
- Persistent storage using AsyncStorage  
- Modern, card-based UI using React Native Paper  
- Fully typed with TypeScript  
- Redux Toolkit for state management
